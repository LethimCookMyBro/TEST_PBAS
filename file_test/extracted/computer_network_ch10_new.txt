=== Page 2 ===
ACL คืออะไร? 
Access Control List (ACL) เป็นกลไกที่ใช้ควบคุมการเข้าถึงเครือข่ายโดยการกรอง แพ็กเก็ต
ที่เข้าหรือออกจากเรา เตอร์หรืออุปกรณ์เครือข่ายอื่น ๆ สามารถใช้ ACL เพื่อก าหนดว่าข้อมูล
ประเภทใดที่สามารถผ่านเครือข่ายได้
=== Page 3 ===
ประเภทของ ACL
1. Standard ACL (หมายเลข 1–99 และ 1300–1999)
•กรองแพ็กเก็ตตามsource IP address เท่านั้น
2. Extended ACL (หมายเลข 100–199 และ 2000–2699)
•กรองแพ็กเก็ตตามsource และ destination IP, protocol (TCP, UDP, ICMP), port number
=== Page 4 ===
รูปแบบการตั้งค่า ACL
1. Number ACL
•Standard -> 1-99
•Extend -> 100-199
2. Name ACL
•ใช้การตั้งชื่อแทนตัวเลข
